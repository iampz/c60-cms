### Shortcuts สำหรับแก้ Formatting
ให้ทำการเลือกข้อความที่ต้องการแล้วกดปุ่มตามด้านล่าง
- Ctrl+U = ขีดเส้นใต้
- Ctrl+Y = ขีดฆ่า
- Ctrl+B = ตัวหนา
- Ctrl+I = ตัวเอียง
